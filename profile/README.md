## 住宅IP VPS，到底和普通VPS有什么区别？

大部分VPS给的是机房IP，也叫数据中心IP。这类IP的特征很明显：用ipinfo.io一查，ASN和company都指向某个IDC服务商，平台一眼就能认出来——"这是服务器，不是真实用户"。

住宅IP VPS就不一样了。它分配的是真实运营商（ISP）给家庭用户用的IP段，通过ipinfo.io查询时，显示的是Comcast、AT&T、Spectrum这类家庭宽带运营商的名字，和你家里装宽带拉出来的IP性质一样。

更进一步的是**双ISP**。双ISP的意思是，这个IP的ASN（自治系统号）和company（所属公司）**都是**真实的家庭宽带运营商，可信度最高，被各平台风控识别为真实用户的概率也最大。

简单类比：机房IP像是公寓楼的公共门牌号，住宅IP像是某个具体的民宅地址，双ISP住宅IP则是连户主信息都对得上的真实民宅地址——平台拦截的时候，当然先拦那个"公寓楼"。
